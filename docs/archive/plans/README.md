# Archived Plans

This directory stores inactive historical plans moved out of the active docs
tree. These files are preserved for context only; they are not the source of
truth for current work.

Current pending work is represented in beads through the `bd` CLI. Current
architecture, command, configuration, and release truth comes from executable
source files such as `Cargo.toml`, `Makefile`, `makefiles/`, `config/*.yaml`,
and the active docs linked from `AGENTS.md`.
