Test constants, fixture values, and timeout defaults.

Documentation: docs/modules/domain.md

Centralized test constants and fixture values for the entire workspace.

All test crates MUST import shared test constants from here instead of defining them locally. This avoids drift between “test-org” in one crate and “org-test” in another.
