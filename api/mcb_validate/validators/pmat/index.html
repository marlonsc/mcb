PMAT Integration Validator (native implementation).

§What

Validates code against PMAT (Property-Based Maintenance Analysis Tool) standards natively within the Rust process, without shelling out to external binaries. Checks include cyclomatic complexity, dead code detection, and technical debt scoring.

§Why

  • Performance: Native implementation avoids process overhead.
  • Integration: Direct access to AST and analysis structures.
  • Compliance: Enforces code quality gates as defined in ADR-036.
