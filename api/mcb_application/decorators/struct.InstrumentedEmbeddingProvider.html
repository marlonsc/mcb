Instrumented embedding provider decorator

Wraps any EmbeddingProvider to add timing metrics collection. All operations are delegated to the inner provider after recording metrics.

§Metrics Recorded

  • Response time (ms) for each operation
  • Success/failure status
  • Cache hits (always false for embeddings, but follows interface)
