Expand description

Generate shared AppContext test infrastructure for a test binary.

Expands to try_shared_app_context(), shared_app_context(), and shared_fastembed_test_cache_dir() functions backed by process-wide OnceLock statics so the ONNX embedding model is loaded only once.

Config is loaded from config/default.toml + env overrides via TestConfigBuilder, then the database path and embedding cache are overridden for test isolation.
