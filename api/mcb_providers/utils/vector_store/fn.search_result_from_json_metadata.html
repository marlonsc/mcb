Build a SearchResult from a JSON metadata/payload object.

Extracts file_path, start_line, content, and language fields using the standard VECTOR_FIELD_* constants. Falls back to line_number when start_line is absent.

Shared across Pinecone, Qdrant, and EdgeVec providers to avoid repeating the same metadata field extraction logic.
