EdgeVec Vector Store Provider

High-performance embedded vector database implementation using EdgeVec. EdgeVec provides sub-millisecond vector similarity search with HNSW algorithm. This implementation uses the Actor pattern to eliminate locks and ensure non-blocking operation.
