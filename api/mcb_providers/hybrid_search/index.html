Hybrid Search Providers

This module provides hybrid search functionality that combines BM25 text-based ranking with semantic (vector) search for improved relevance.

§BM25 Algorithm

BM25 (Best Matching 25) scores documents based on term frequency and inverse document frequency:

  • High scores: Documents with query terms that are rare in the corpus
  • Low scores: Documents with common terms or no matching terms

Parameters:

  • k1: Term frequency saturation (default: 1.2)
  • b: Document length normalization (default: 0.75)
