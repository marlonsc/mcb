SQLite backend for the memory and full project schema.

Documentation: docs/modules/providers.md

Implements SchemaDdlGenerator (full project: collections, observations, session_summaries, file_hashes) for SQLite. Provides SqliteExecutor (port DatabaseExecutor), SqliteMemoryRepository (port MemoryRepository), and factory functions for DI.
