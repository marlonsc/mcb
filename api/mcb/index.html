MCB binary crate.

Command-line entry point and re-export surface for the Memory Context Browser. Supports runtime server modes (HTTP and stdio) plus management subcommands.

§Typical usage

mcb serve --transport http
mcb serve --transport stdio

Re-exports§

pub use crate::loco_app::McbApp;
