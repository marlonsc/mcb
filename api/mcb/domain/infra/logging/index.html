Logging facade: set_log_fn + log_operation (macros dispatch here).

Logging facade: single registration point for the operation logger.

Infra (or tests) call set_log_fn at startup so that macros trace! .. error! forward to the real implementation. Domain has no dependency on tracing or infra.
