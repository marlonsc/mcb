Workflow FSM entities for session state management Workflow FSM entities for session state management and transitions.

Documentation: docs/modules/domain.md

This module implements the finite state machine (FSM) for workflow orchestration. It defines the various states, transition triggers, and audit records required to manage the lifecycle of an agent-led workflow session.
