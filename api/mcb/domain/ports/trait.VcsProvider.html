Expand description

Version Control System provider for repository operations.

Abstraction over version control systems (Git, Mercurial, SVN, etc.). The current implementation focuses on Git, but the trait is designed to support other VCS implementations in the future.

Discover and list repositories under a root path

Scans the given root directory for VCS repositories (e.g., directories containing .git). Returns a list of discovered repositories.

§Arguments
  • root - Root directory to scan for repositories
§Returns

Vector of discovered repositories, or empty vector if none found.
