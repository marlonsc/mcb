1//! Vector Store Provider Implementations
2//!
3//! **Documentation**: [docs/modules/providers.md](../../../../docs/modules/providers.md#vector-store-providers)
4//!
5//! Provides storage backends for vector embeddings.
6//!
7//! ## Available Providers
8//!
9//! | Provider | Type | Description |
10//! | ---------- | ------ | ------------- |
11//! | EdgeVecVectorStoreProvider | Embedded | High-performance HNSW vector store (local) |
12//! | EncryptedVectorStoreProvider | Secure | AES-256-GCM encryption wrapper |
13//! | MilvusVectorStoreProvider | Cloud | Production-scale cloud vector database |
14//! | PineconeVectorStoreProvider | Cloud | Pinecone cloud vector database |
15//! | QdrantVectorStoreProvider | Cloud | Qdrant vector search engine |
16//!
17//! ## Provider Selection Guide
18//!
19//! - **Development/Testing**: Use `EdgeVecVectorStoreProvider` for local HNSW storage
20//! - **Production with encryption**: Use `EncryptedVectorStoreProvider` wrapper
21//! - **Cloud production**: Use `MilvusVectorStoreProvider` or `QdrantVectorStoreProvider`
