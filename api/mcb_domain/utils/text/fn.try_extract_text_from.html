pub fn try_extract_text_from<T: Serialize>(
    content: &[T],
) -> Result<String, Error>
Expand description

Extract text from any serializable MCP content sequence, returning a Result to surface serialization failures to callers.

Serializes the content to JSON, then extracts "text" fields. This allows callers to pass protocol-specific types (e.g. rmcp::model::Content) without the domain layer depending on those types directly.

§Errors

Returns serde_json::Error if any element in content fails to serialize to a JSON value.
