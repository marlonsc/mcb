Expand description

Cryptographic provider port

Defines the contract for encryption/decryption operations. Implementations provide the actual cryptographic primitives (e.g., AES-256-GCM).

§Example

use mcb_domain::ports::{CryptoProvider, EncryptedData};
use std::sync::Arc;

fn encrypt_metadata(crypto: Arc<dyn CryptoProvider>, data: &[u8]) -> mcb_domain::Result<EncryptedData> {
    crypto.encrypt(data)
}
