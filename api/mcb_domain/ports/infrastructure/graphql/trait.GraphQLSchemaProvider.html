Expand description

Domain port for GraphQL schema providers.

Implementations build a GraphQL schema from the database connection and return it as a type-erased Box<dyn Any + Send + Sync>. The concrete schema type (e.g. async_graphql::dynamic::Schema) lives in the provider layer; the domain remains free of GraphQL dependencies.
