Expand description

Language-Specific Code Chunking Provider

Defines the contract for providers that parse and chunk source code in a specific programming language. Each provider uses AST-based parsing to extract semantically meaningful code segments.

§Implementations

The system supports multiple language providers:

  • Rust, Python, JavaScript, TypeScript
  • Go, Java, C, C++, C#
  • Ruby, PHP, Swift, Kotlin

§Example

use mcb_domain::ports::LanguageChunkingProvider;
use std::sync::Arc;

fn chunk_code(provider: Arc<dyn LanguageChunkingProvider>) {
    // Chunk source code
    let source = "fn main() { println!(\"Hello\"); }";
    let chunks = provider.chunk(source, "src/main.rs");
    println!("Extracted {} chunks from {}", chunks.len(), provider.provider_name());
}
