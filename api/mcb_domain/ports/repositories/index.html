Repository ports for data persistence Repository ports for data persistence.

All repository traits use #[async_trait] and require Send + Sync. Standard CRUD traits are generated by define_crud_port! / define_readonly_port! macros to eliminate boilerplate.

Organized into submodules by domain aggregate.
